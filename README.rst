----

A drop-in replacement for django's ``ImageField`` that provides a flexible, intuitive and easily-extensible interface for creating new images from the one assigned to the field.

`Click here <http://django-versatileimagefield.readthedocs.org/en/latest/in_a_nutshell.html>`_ for a quick overview of what it is, how it works and whether or not it's the right fit for your project.
